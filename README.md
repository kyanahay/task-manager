# Task Manager App

A simple and interactive task manager built with Next.js and React. This project demonstrates core React concepts including state management, component structure, and side effects.

## Features

* Add new tasks
* Mark tasks as complete/incomplete
* Delete tasks
* Filter tasks (All / Active / Done)
* Persist tasks using localStorage

## Tech Stack

* Next.js (App Router)
* React (useState, useEffect)
* Tailwind CSS

## How to Run

```bash
npm install
npm run dev
```

Then open:
http://localhost:3000

## Project Structure

* TaskBoard → manages state and logic
* AddTaskForm → handles user input
* TaskList → renders task list
* TaskCard → displays individual tasks

## Concepts Demonstrated

* Component-based architecture
* Props and one-way data flow
* State management with useState
* Side effects with useEffect
* Conditional rendering
* Derived state (task filtering)

## Notes

This project was built as part of a React/Next.js learning module to demonstrate core frontend development concepts.
